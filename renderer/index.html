AI — команды инструментов
Sirius AI может управлять файлами. Просто попроси его, например: «Создай файл main.py с кодом...». Он автоматически использует команды:
[CREATEFILE:путь]Создать файл
[EDITFILE:путь]Изменить файл
[READFILE:путь]Прочитать файл
[READDIR:путь]Список файлов в папке
[DELETEFILE:путь]Удалить файл
Примеры запросов к AI
"Создай Flask-приложение с роутами /api/users"
"Прочитай main.py и найди баги"
"Добавь TypeScript типы в index.ts"
"Создай структуру проекта React"
"Покажи что находится в папке src"
Подключение LM Studio
1. Открой LM Studio и загрузи любую модель
2. Перейди в раздел Local Server
3. Нажми Start Server (порт 1234)
4. Индикатор в правом нижнем углу станет зелёным
Файловый менеджер (ПКМ)
Правый клик на файле или папке в дереве открывает меню: создать, переименовать, удалить.
Терминал
Встроенный терминал поддерживает несколько вкладок. Кнопка + создаёт новый терминал. Работает PowerShell/CMD на Windows, bash на Linux/macOS.

Тема редактора
Подсветка синтаксиса и фон редактора кода.
Системный промпт (роль модели)
Кратко задаёт роль и стиль ответов. К нему автоматически добавляются: инструкции по инструментам (CREATEFILE, EDITFILE, READFILE, READDIR, DELETEFILE), контекст проекта и открытый файл.
Подсказка
Инструкции по файловым командам и контекст проекта подставляются автоматически. Здесь задаётся только «личность» модели (например: «Фокус на Python и чистом коде»).
